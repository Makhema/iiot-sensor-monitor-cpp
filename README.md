# 🌐 iiot-sensor-monitor-cpp

📌Overview

This project is an Industrial Internet of Things(IIoT) sensor monitoring system developed in C++. It simulates the management and processing of sesor data collected from multiple devices arranged in a 2D grid. The system reads sensor data from a text file, processes active sensor readings, and exports results to both text and binary files.
